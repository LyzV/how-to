Как запустить TFTP загрузку в целевую плату:
1. Настроить локальную сеть:
    - Найти Ehernet-интерфейс, который подключен к плате. Именно его и будем настраивать.
    - Настройку выполняем через GUI: Settings->Network. Для интерфейса Ehernet(enp0s8) 
      выбрать шестерёночку.
    - Выбрать вкладку IPv4. Выбрать ручную настройку интерфейса - Manual.
      задать: адрес- 192.168.3.10, маску 255.255.255.0, шлюз- 192.168.3.10.
    - Нажать Apply.
2. Установить TFTPD сервер на виртуальную машину:
    - sudo apt-get -y install tftpd-hpa xinetd
3. Сделать папку загрузки:
    - sudo mkdir /tftpboot
    - sudo chmod -R 777 /tftpdboot
4. Запустить сервер:
    - sudo service tftpd-hpa restart
5. Проверить работу сервера:
    - sudo service tftpd-hpa status
6. Разместить текстовый файл hello.txt в папке загрузки /tftpboot
6. На целевой плате проверить связь с хостом:
    - ping 192.168.3.10
7. На целевой плате выполнить загрузку в текущую директорию текстового файла:
    - tftp -g -r hello.txt 192.168.3.10
