

0) Все элементы модели доступны только через временный индекс.

1) Индекс можно получить по строке, колонке и родительскому индексу.
model->index(y, x, QModelIndex);

2) По индексу всегда можно получить модель: 
QAbstractItemModel *model = index.model();

3) Элементы модели могут иметь различные роли:
	a) DisplayRole - данные в форме строки для отображения элемента;
	б) EditRole - данные в форме строки, пригодные для редактирования;
	в) StatusTipRole - данные в форме строки для отображения в статус-баре;
	г) DecorationRole - иконка для символического представления элемента слева от отображения.

4) QHeaderView - представление, которое так-же обращается к модели.

5) У представления есть сигнал activated. От возникает, когда нажали ввод на элементе отображения.
   Это можно использовать для понимания модели о навигации оператора.

6) На одно представление устанавливается один делегат. 

7) Для того чтобы модель сделать редактируемой:
	- реализовать функцию flags;
	- реализовать функцию setData;
	- поддержать редактирование в функции data.
